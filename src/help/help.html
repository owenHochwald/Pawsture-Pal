Notification Troubleshooting

If you're having trouble with notifications, here are a few common issues and solutions:

First, check if notifications are enabled in Chrome. Some extensions can block notifications, or they might be turned off in your system settings. To fix this:

1. Make sure Chrome notifications are enabled in your system settings (Windows Settings > System > Notifications)
2. Check if any other extensions might be blocking notifications
3. Try disabling and re-enabling the extension

Sometimes simply restarting Chrome can also resolve notification issues.
If you're using Windows 11, make sure Focus Assist isn't blocking notifications. You can check this in Windows Settings > System > Focus Assist.

Disable Notifications

To temporarily stop receiving posture reminders, you have two options:
1. Quick Disable: Right-click the extension icon and select "Manage Extension" > Toggle off the switch
2. Set Long Interval: Open the extension popup and set a very long interval (e.g., 999 minutes)
You can re-enable the extension anytime using the same steps.
