Not Receiving Notifications ‚ñº

If you're not receiving notifications, try these steps:

1. Check Chrome Notification Settings:
  • Open Chrome Settings (‚ãÆ menu > Settings)
  • Go to "Privacy and security" > "Site Settings" > "Notifications"
  • Make sure Chrome is allowed to send notifications in your system
  • Find "Pawsture Pal" and ensure it's set to "Allow"
2. Check System Notifications:
  • Windows: Settings > System > Notifications
  • Make sure notifications are enabled for Chrome
Note: After changing settings, you might need to restart Chrome for changes to take effect.
Notification Buttons Not Working ‚ñº

If notification buttons aren't responding:

  • Make sure Chrome is up to date
  • Try disabling and re-enabling the extension
  • Check if other Chrome notifications are working
Tip: Sometimes a simple Chrome restart can fix button interaction issues!
Notifications Stop After Sleep/Hibernate ‚ñº

If notifications stop after your computer wakes up:

  • Click the extension icon to open the popup
  • The extension should automatically reconnect
  • If not, try clicking the "Check Now" button
Chrome sometimes pauses background processes during sleep. Opening the popup helps restart them.
Wrong Notification Timing ‚ñº

If notifications are coming at wrong intervals:

  • Check your interval setting in the extension popup
  • Make sure Chrome is not in power saving mode
  • Try setting a slightly longer interval (at least 1 minute)
Chrome may adjust notification timing to save battery. Using longer intervals helps maintain consistency.
Still Having Issues? ‚ñº

If you're still experiencing problems:

  • Try removing and reinstalling the extension
  • Clear Chrome's cache and cookies
  • Check if your antivirus is blocking notifications
  • Make sure Chrome has system permissions
